  LOFT Beauty CRM

Веб-приложение для управления салоном красоты.

  Возможности

- Просмотр услуг
- Просмотр мастеров
- Онлайн-запись
- Управление клиентами
- Управление услугами
- Управление мастерами
- Управление записями
- Отчеты

   Технологии

- ASP.NET Core Razor Pages
- Entity Framework Core
- SQL Server
- Bootstrap
