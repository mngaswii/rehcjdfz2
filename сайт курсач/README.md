# \# LOFT Beauty CRM

# 

# Веб-приложение для управления салоном красоты, разработанное на ASP.NET Core Razor Pages и Entity Framework Core.

# 

# \## Возможности

# 

# \### Для клиентов

# \- Просмотр услуг

# \- Просмотр мастеров

# \- Онлайн-запись на процедуры

# 

# \### Для администратора

# \- Управление клиентами

# \- Управление мастерами

# \- Управление услугами

# \- Управление записями

# \- Управление платежами

# \- Просмотр отчетов

# 

# \## Технологии

# 

# \- ASP.NET Core Razor Pages

# \- Entity Framework Core

# \- SQL Server

# \- Bootstrap 5

#
